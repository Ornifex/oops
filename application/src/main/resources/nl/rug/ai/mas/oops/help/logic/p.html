Propositional logic

Ordinary propositional logic is not a modal logic, but is also supported because it forms the basis for all the modal logics supported by OOPS.

Frame properties

Because propositional logic is not a modal logic, formally it does not have Kripke semantics. For the purpose of uniformity, we can consider a Kripke model to be a propositional model if it has a single world and all relations are empty.
